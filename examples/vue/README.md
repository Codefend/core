# Codefend Vue example

## Usage

```bash
npm i -g codefend@latest // or npm i -D codefend@latest
codefend -i
codefend -o
cd codefend-output
code .
```

### Output

The code generated can be seen in [`codefend-output`](https://github.com/Codefend/core/tree/main/examples/vue/codefend-output)
