# Codefend Vue example

## Usage

```bash
npm i -g codefend@latest // or npm i -D codefend@latest
codefend -i
codefend -o
cd codefend-output
code .
```

### Output

The code generated can be viewed inside [`codefend-output`](./codefend-output) folder
